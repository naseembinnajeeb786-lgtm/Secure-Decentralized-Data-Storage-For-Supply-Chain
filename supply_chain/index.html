What is this project?

A simulation of how real-world supply chains can use blockchain technology to create an immutable, tamper-proof record of every product journey.

Blockchain Ledger
A chain of records where each block holds supply chain data and is cryptographically linked to the previous block using its SHA-256 hash.
SHA-256 Hashing
Every block generates a unique 64-character fingerprint from its data. Change even one character and the fingerprint changes completely — fraud detected instantly.
Tamper Detection
Modify any historical record and the chain validation immediately detects the mismatch. No data can be secretly altered — making it truly immutable.
Supply Chain Events
Tracks a product journey: Raw Materials → Manufacturing → Warehouse → Delivery. Each stage is a permanent, verifiable block in the chain.

How It Works

Four simple concepts that make a blockchain secure and immutable.

01
Data is entered
A supply chain event (location, quantity, event type) is added as a new block to the chain.
02
Hash is calculated
SHA-256 converts the block's data + previous hash into a unique 64-character fingerprint.
03
Blocks are linked
Each new block stores the previous block's hash — creating a chain. Change one block, and the link breaks.
04
Validation runs
is_valid() checks every block's hash matches its data AND that every link to the previous block is intact.

Python Source Code

Built with only Python built-in libraries — no external packages needed.

import hashlib import datetime class Block: def __init__(self, index, data, previous_hash): self.index = index self.timestamp = str(datetime.datetime.now()) self.data = data self.previous_hash = previous_hash self.hash = self.calculate_hash() def calculate_hash(self): # Combine all fields → SHA-256 → 64-char fingerprint text = str(self.index) + self.timestamp + \ str(self.data) + self.previous_hash return hashlib.sha256( text.encode() ).hexdigest()
class Blockchain: def __init__(self): self.chain = [Block(0, "Genesis Block", "0")] def add_block(self, data): prev = self.chain[-1] new = Block(len(self.chain), data, prev.hash) self.chain.append(new) def is_valid(self): for i in range(1, len(self.chain)): curr = self.chain[i] prev = self.chain[i - 1] if curr.hash != curr.calculate_hash(): return False if curr.previous_hash != prev.hash: return False return True
# Normal valid chain sc = Blockchain() sc.add_block({"event": "Warehouse", "qty": 480}) print(sc.is_valid()) # True # Hacker tampers with Block 1 sc.chain[1].data = {"event": "Warehouse", "qty": 200} print(sc.is_valid()) # False — caught!
